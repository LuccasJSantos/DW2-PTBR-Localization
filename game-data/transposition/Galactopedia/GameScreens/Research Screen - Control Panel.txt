À direita da tela está o Painel de Controle de Pesquisa. Este painel oferece várias funções úteis para navegar na árvore tecnológica.

[H3]Saindo da Pesquisa[/H3]
Em cima fica o botão "Sair da Pesquisa". Clique neste botão para fechar a tela de pesquisa e voltar ao jogo.

[H3]Projetos Concurrentes Máximos[/H3]
Abaixo disso está uma explicação sobre o número máximo de projetos de pesquisa concorrentes. Passe o cursor sobre esta explicação para obter mais detalhes.

[H3]Zoom In/Out[/H3]
À direita ficam os botões "Zoom In" e "Zoom Out", que permitem ver mais ou menos dos projetos disponíveis.

[H3]Botões de Pulo[/H3]
Em seguida vêm os botões de pulo. Clicando em um desses botões permite que você se mova instantaneamente para a primeira linha de projetos dessa categoria.

Por exemplo, clicar no botão "Tecnologia de Hiperespaço" fará com que a vista seja movida instantaneamente para mostrar os projetos de hiperdrive no topo da tela.

[H3]Filtrando Projetos[/H3]
Abaixo disso está uma lista suspensa que permite filtrar os projetos mostrados. Isso permite que você se concentre exclusivamente em uma categoria. É claro que também significa que você não poderá ver os outros projetos. Portanto, lembre-se de voltar para "Todos os Projetos".

[H3]Próximo Projeto Sugerido[/H3]
Em seguida está o botão "Adicionar Projeto Sugerido à Fila". Este botão sugere o próximo projeto que seus consultores em pesquisa acham que você deveria pesquisar. Clique no botão para adicionar o projeto sugerido à fila.

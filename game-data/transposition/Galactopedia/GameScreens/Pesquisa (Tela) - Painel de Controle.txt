À direita da tela está o Painel de Controle de Pesquisa. Este painel fornece várias funções úteis para navegar pela árvore tecnológica.

[H3]Saindo da Pesquisa[/H3]
Na parte superior está o botão Sair da Pesquisa. Clique neste botão para fechar a tela de Pesquisa e retornar ao jogo.

[H3]Máximo de Projetos Concorrentes[/H3]
Abaixo está uma explicação sobre o número máximo de projetos de pesquisa concorrentes. Passe o mouse sobre esta explicação para obter mais detalhes.

[H3]Zoom In/Out[/H3]
À direita estão os botões de Aumentar e Reduzir Zoom que permitem ver mais ou menos dos projetos disponíveis.

[H3]Botões de Salto[/H3]
Em seguida estão os botões de Salto. Clicando em um desses botões você pode mover imediatamente a visualização para a primeira linha de projetos daquela categoria.

Por exemplo, clicando no botão 'Tecnologia Hiperspacial' você moverá imediatamente a visualização para mostrar os projetos de hipermotivação na parte superior da tela.

[H3]Filtrando Projetos[/H3]
Abaixo disso há uma lista suspensa que permite filtrar os projetos exibidos. Isso permite que você se concentre exclusivamente em uma categoria. Claro que isso também significa que você não poderá ver os outros projetos. Portanto, certifique-se de retornar para 'Todos os Projetos'.

[H3]Próximo Projeto Sugerido[/H3]
A seguir está o botão 'Fila de Projeto Sugerido'. Este botão sugere o próximo projeto que seus conselheiros de pesquisa acham que você deve pesquisar. Clique no botão para colocar o projeto sugerido na fila.

Passar o mouse sobre o botão fornecerá um resumo do projeto sugerido. Enquanto passa o mouse, o projeto também será 'pingado' na visualização (você pode precisar dar zoom para ver onde está).

[H3]Fila de Pesquisa[/H3]
Finalmente, há a fila de pesquisa. Estes são todos os projetos que estão atualmente na fila para pesquisa. Os projetos mais altos estão sendo ativamente progredidos (o número de projetos concorrentes), enquanto os projetos subsequentes estão aguardando até que os projetos principais sejam concluídos.

Você pode remover projetos da fila clicando com o botão direito sobre eles na fila. Você também pode mover projetos para cima ou para baixo na fila usando as setas à direita.

Componente Disponíveis

A lista de componentes disponíveis exibe todos os componentes pesquisados que podem ser adicionados ao projeto.

São exibidos apenas os componentes válidos para o papel. Assim, lutadores não veem nenhum componente de nave e vice-versa. Os componentes também serão excluídos se forem muito grandes para qualquer baía no casco.

Você pode filtrar ainda mais os componentes usando a lista suspensa no canto superior esquerdo do painel. Isso permite filtrar por tipo de baía ou mostrar apenas o último componente por categoria.

Você pode classificar os componentes usando a lista suspensa no canto superior direito do painel. Isso permite classificação por nome, categoria, nível tecnológico ou tamanho.

Resumo do Componente
Ao passar o mouse sobre cada componente, uma janela pop-up aparecerá à direita, descrevendo as especificações do componente.

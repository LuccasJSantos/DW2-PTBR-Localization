À direita da tela está o Painel de Controle de Pesquisa. Este painel oferece várias funções úteis para navegar na árvore tecnológica.

[H3]Sair da Pesquisa[/H3]
No topo está o botão Sair da Pesquisa. Clique neste botão para fechar a tela de Pesquisa e retornar ao jogo.

[H3]Número Máximo de Projetos Simultâneos[/H3]
Abaixo disso está uma explicação sobre o número máximo de projetos de pesquisa simultâneos. Passe o mouse sobre essa explicação para obter mais detalhes.

[H3]Zoom In/Out[/H3]
À direita estão os botões de Ampliar e Reduzir que permitem ver mais ou menos dos projetos disponíveis.

[H3]Botões de Salto[/H3]
Em seguida estão os botões de Salto. Clicar em um desses botões permite mover imediatamente a visualização para a primeira linha de projetos dessa categoria.

Por exemplo, clicar no botão de salto 'Tecnologia de Hiperespaço' moverá imediatamente a visualização para mostrar os projetos de hiperpropulsão na parte superior da tela.

[H3]Filtragem de Projetos[/H3]
Abaixo disso há uma lista suspensa que permite filtrar os projetos exibidos. Isso permite que você se concentre exclusivamente em uma categoria. No entanto, isso também significa que você não verá os outros projetos. Certifique-se de voltar para 'Todos os Projetos'.

[H3]Próximo Projeto Sugerido[/H3]
Em seguida está o botão 'Colocar Projeto Sugerido na Fila'. Este botão sugere o próximo projeto que seus conselheiros de pesquisa acham que você deveria pesquisar. Clique no botão para colocar o projeto sugerido na fila.

Passe o mouse sobre o botão para obter um resumo do projeto sugerido. Enquanto o mouse estiver sobre o botão, o projeto também será destacado na visualização (talvez seja necessário ampliar a visão para ver onde ele está).

[H3]Fila de Pesquisa[/H3]
Finalmente, há a fila de pesquisa. Estes são todos os projetos que estão atualmente na fila para pesquisa. Os projetos mais no topo estão sendo progressivamente ativos (o número de projetos simultâneos), enquanto os projetos subsequentes estão esperando até que os projetos no topo sejam concluídos.

Você pode remover projetos da fila clicando com o botão direito sobre eles na fila. Você também pode mover projetos para cima ou para baixo na fila usando as setas à direita.

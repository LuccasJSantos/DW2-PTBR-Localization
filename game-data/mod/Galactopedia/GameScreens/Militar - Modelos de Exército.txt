Os Modelos de Exército permitem que você defina a composição relativa dos tipos de tropas. Os Modelos de Exército são utilizados por frotas e colônias para especificar quantos de cada tipo de tropa devem ter.

[H3]Tipos de Modelo de Exército[/H3]
Existem 4 tipos básicos de Modelo de Exército:
[ULI]Invasão: atribuído a frotas para definir quais tropas devem ser usadas ao invadir colônias inimigas. Tipicamente incluem muito Armadura, algumas Forças Especiais, alguma Infantaria e Titãs (se disponíveis)[/ULI]
[ULI]Defesa Major de Colônia: atribuído a colônias principais (como sua capital) para definir quais tropas devem ser usadas para defender a colônia de tropas inimigas invasoras. Tipicamente incluem todos os tipos de tropas, especialmente muita infantaria e unidades de defesa planetária.[/ULI]
[ULI]Defesa de Colônia de Fronteira: atribuído a colônias vulneráveis na fronteira para definir quais tropas devem ser usadas para defender a colônia de tropas inimigas invasoras. Tipicamente incluem muita infantaria e algumas unidades de defesa planetária.[/ULI]
[ULI]Defesa Padrão de Colônia: atribuído a todas as outras colônias para definir quais tropas devem ser usadas para defender a colônia de tropas inimigas invasoras. Tipicamente incluem muita infantaria.[/ULI]

Existe um modelo padrão para cada tipo, mas você pode modificá-los ou criar modelos adicionais que atendam às suas necessidades.

[H3]Atribuindo Modelos de Exército[/H3]
Você pode selecionar um Modelo de Exército para uma colônia selecionando a colônia e usando os botões 'Automatizar Colônia' > 'Modelo de Exército' sob o Painel de Seleção. Isso abrirá uma lista de todos os Modelos de Exército que podem ser usados. Selecione o modelo que deseja usar para a colônia clicando nele na lista. A colônia então tentará recrutar tropas defensivas na proporção estabelecida pelo modelo.

Você pode selecionar um Modelo de Exército para uma frota selecionando a frota e usando o botão 'Modelo de Exército' sob o Painel de Seleção. Isso abrirá uma lista de todos os Modelos de Exército que podem ser usados. Selecione o modelo que deseja usar para a frota clicando nele na lista. A frota então tentará carregar tropas na proporção estabelecida pelo modelo.

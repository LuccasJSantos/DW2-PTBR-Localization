Passar o mouse sobre um projeto apresentará um resumo do projeto, incluindo os benefícios que desbloqueia. Conforme você mantém o cursor sobre o projeto por mais tempo, um resumo mais detalhado é exibido, com dados abrangentes sobre os benefícios do projeto.

Observe que você também pode passar o mouse sobre os projetos na fila de pesquisa e sobre o botão 'Adicionar Projeto Sugerido à Fila'.

[H3]Adicionar um Projeto à Fila de Pesquisa[/H3]
Clique com o botão esquerdo em um projeto para adicioná-lo à fila de pesquisa.

Clique com o botão direito em um projeto para removê-lo da fila de pesquisa.

[H3]Custos de Iniciação[/H3]
Alguns projetos requerem um custo de iniciação a ser pago antes que a pesquisa possa começar. O custo pode incluir dinheiro e recursos.

Quando você clica com o botão esquerdo para adicionar à fila um projeto que possui custos de iniciação, uma janela de prompt aparecerá pedindo sua aprovação para o pagamento dos custos.

[H3]Requisitos Extras[/H3]
Alguns projetos têm requisitos extras de bônus de pesquisa antes que você possa começar a pesquisá-los. Isso significa que eles só podem ser iniciados quando você tem bônus de pesquisa suficientes de um tipo específico, por exemplo, alguns projetos avançados de escudo exigem que você tenha um certo limite de bônus de pesquisa de escudo.

Você pode obter bônus de pesquisa em áreas específicas procurando por esses locais através da exploração. Quando encontrar uma área com bônus de pesquisa especiais, você pode então construir uma estação de pesquisa lá para obter esses bônus para seu império.

[H3]Pesquisa Acelerada[/H3]
Você pode acelerar a pesquisa de um projeto importante através da 'pesquisa acelerada'. Isso significa gastar dinheiro para dobrar a taxa de pesquisa desse projeto.

Para pesquisar aceleradamente um projeto:
[ULI]certifique-se de que o projeto está sendo atualmente pesquisado, ou seja, é o projeto principal na fila[/ULI]
[ULI]clique no projeto na visualização[/ULI]
[ULI]uma janela de prompt aparecerá pedindo sua aprovação para o pagamento para pesquisa acelerada do projeto[/ULI]
